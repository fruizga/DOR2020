Welcome

This Project is a computer technology related to computer vision and image processing that detects and defines objects. This program has the ability to classify just one or several objects within a digital image at once, the main usage is to select and classify ripe fruits by color. Artificial Intelligence has been around for years, but it is becoming more apparent across a range of industries now more than ever before.

Usage



  1. Please click on "Choose file" button
  2. Make sure to upload clear pictures. For a more accurate experience JPG format would be ideal.
  3. click on the button "Analyze".
